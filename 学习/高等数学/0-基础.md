

### 三角函数

| | sin | cos | tan | 
| -- | -- | -- | -- |
| 0 | 0 | 1 | 0 |
| $\frac{π}{6} $ | $\frac{1}{2}$ | $\frac{√3}{2}$ | $\frac{√3}{3}$ |
| $\frac{π}{4}$ | $\frac{√2}{2}$ | $\frac{√2}{2}$ | $1$ |
| $\frac{π}{3}$ | $\frac{√3}{2}$ | $\frac{1}{2}$ | $√3$ |
| $\frac{π}{2}$ | 1 | 0 | 不存在 |
| $π$ | 0 | -1 | 0 |
| $2π$ | 0 | 1 | 0 |

- $arctan(-∞) = -\frac{π}{2}$
- $arctan(+∞) = \frac{π}{2}$

- 在 $(0,\frac{π}{2})$ 内， $sinx < x < tanx$
